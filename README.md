# 🧊 Glassmorphic Icons on Color Tiles

Glassmorphic Icons on Color Tiles is a sleek and modern UI component that blends transparent, frosted glass style icons with bold and colorful background tiles. Ideal for dashboards, portfolios, mobile apps and modern web interfaces.

---

## 🚀 Features

- 🧊 **Glassmorphism Aesthetic** : Smooth frosted glass effect on icons for a futuristic look.
- 🎨 **Vibrant Color Tiles** : Each icon sits on a unique and vivid color block for visual contrast.
- 💡 **Minimal and Modern Design** : Clean layout that enhances readability and user focus.
- 📱 **Responsive Layout** : Fully adaptable to different screen sizes and devices.
- ⚙️ **Easy Customization** : Change icon sets, colors, effects or layout with simple CSS tweaks.
- 💻 **Pure HTML and CSS** : Lightweight and dependency free, built with semantic HTML and modern CSS.
- 🌐 **Cross Browser Compatible** : Consistent rendering across all major browsers.
